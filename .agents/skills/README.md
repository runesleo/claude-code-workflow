# Kimi Code Skills

This directory contains Vibe workflow skills for Kimi Code.

## Usage

```bash
# List available skills
kimi skill list

# Run a specific skill
kimi skill run session-end
```

## Available Skills

- `systematic-debugging` — Find root cause before attempting fixes.
- `verification-before-completion` — Require fresh verification evidence before claiming completion.
- `session-end` — Capture handoff, memory, and wrap-up state before ending a session.
