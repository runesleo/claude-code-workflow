## 🚨 常见问题处理

### 配置验证失败
**症状**: Schema验证报错  
**解决**:
1. 检查YAML格式
2. 验证必需字段
3. 检查字段类型
4. 参考已迁移平台配置

### 测试失败
**症状**: 单元测试或集成测试失败  
**解决**:
1. 检查配置是否正确加载
2. 验证生成文件路径
3. 检查文档类型配置
4. 对比已通过平台

### 生成文件不正确
**症状**: 生成的文件内容或结构错误  
**解决**:
1. 检查 output_paths 配置
2. 验证 doc_types 配置
3. 检查特殊处理钩子
4. 手动验证模板渲染

### 性能回归
**症状**: 生成速度明显变慢  
**解决**:
1. 检查YAML缓存
2. 优化文件操作
3. 减少不必要的复制
4. 使用性能分析工具
